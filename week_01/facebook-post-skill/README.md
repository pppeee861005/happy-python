# Facebook 智慧發文 Skill

## 📱 概述

這是一個 Claude AI Skill，用來協助用户在 Facebook 上自動發佈文章。透過與 Claude in Chrome 工具的整合，它能夠：

- ✨ 自動操作 Facebook 發文流程
- 📝 幫助組織和優化文章內容
- 🎯 支援多種發文場景（介紹、分享、公告等）
- 🔐 安全可靠地發佈你的內容

## 🚀 快速開始

### 安裝

1. **下載 Skill**：將 `facebook-post-skill` 目錄複製到你的 Skills 資料夾
2. **驗證安裝**：確保 Claude 能夠識別此 Skill
3. **測試連接**：確保 Claude in Chrome 擴展已安裝

### 基本用法

```
你：幫我在 Facebook 上發佈這篇文章：
[你的文章內容]

Claude 會：
✅ 確認內容無誤
✅ 打開你的 Facebook 帳戶
✅ 自動發佈文章
✅ 顯示發佈成功的截圖
```

## 🔄 工作流程

### 簡化流程圖

```
用戶輸入
    ↓
判斷：文本 vs 主題
    ├─→ 文本：確認後直接發佈
    └─→ 主題：撰寫 → 確認 → 發佈
         ↓
打開 Facebook
    ↓
自動化操作
  1. 點擊發文框
  2. 輸入文章
  3. 點擊發佈
    ↓
確認成功
    ↓
顯示結果
```

### 詳細步驟

1. **內容準備**：接收或撰寫文章內容
2. **內容確認**：向用戶展示最終文本並確認
3. **瀏覽器初始化**：驗證 Facebook 已登入
4. **元素定位**：使用 find 工具定位發文框和發佈按鈕
5. **自動輸入**：點擊、輸入、發佈
6. **成功確認**：等待並拍攝結果截圖

## 🐛 常見問題

### Q: Chrome 擴展無法連接
**A**: 確保 Chrome 已打開並且擴展已啟用。如果仍無法連接，嘗試重新授予權限。

### Q: 找不到發文框
**A**: 確保 Facebook 頁面已完全加載。如果使用不同語言的 Facebook，查詢詞可能需要調整。

### Q: 文章未發佈
**A**: 檢查文本框中的內容是否完整，確認「發佈」按鈕可見，並確保網路連接穩定。

### Q: 無法登入 Facebook
**A**: 在瀏覽器中手動登入 Facebook，然後重試。此 Skill 使用已有的登入狀態。
