# Facebook 智慧發文 Skill

## 概述

此技能協助你在 Facebook 上智慧發佈文章。它能夠：

- 📱 **自動操作流程**：利用 Chrome 瀏覽器和 Claude in Chrome 工具，自動完成點擊、輸入和發佈
- ✏️ **內容組織**：幫助優化和組織文章內容，增加表情符號和格式
- 🎯 **多種用途**：支持發佈一般貼文、技術分享、產品介紹、新聞公告等
- 📝 **靈活輸入**：支持用戶直接提供完整文本，或提供主題讓技能協助組織內容

## 使用方式

### 準備工作

1. **打開 Facebook**：在 Chrome 瀏覽器中打開 Facebook 並確保已登入
2. **確認 Chrome 擴展**：確保 Claude in Chrome 擴展已安裝並連接

### 發佈文章的步驟

#### 方式 A：用戶提供完整文本

```
你：幫我在 Facebook 上發佈這篇文章：[你的完整文本]
```

Claude 會：
1. 確認文本內容無誤
2. 打開瀏覽器並截圖確認 Facebook 已登入
3. 點擊發文區域
4. 將文本輸入到文字框
5. 點擊「發佈」按鈕
6. 確認發佈成功並顯示結果

#### 方式 B：用戶提供主題，Claude 協助組織內容

```
你：幫我寫一篇介紹 OpenCLI 的 Facebook 文章，要包括特點、使用範例和署名
```

Claude 會：
1. 根據主題生成結構化內容
2. 確認無誤後進行發佈
3. 顯示最終發佈的文章

## 核心工作流

### 第 1 步：獲取文章內容
- 如果用戶已提供完整文本 → 直接跳到第 3 步
- 如果用戶只提供主題 → 協助組織和撰寫內容

### 第 2 步：內容優化
優化考慮因素：
- ✨ 適當使用表情符號
- 📋 清晰的段落結構
- 🎯 吸引眼球的開頭
- 📌 清晰的結論或行動呼籲
- 🔗 如需要，加入簽名或來源註明

### 第 3 步：打開瀏覽器並確認登入
```
步驟：
1. 截圖檢查瀏覽器狀態
2. 確認用戶已登入 Facebook
3. 使用 find 工具定位發文輸入框
```

### 第 4 步：自動填寫和發佈
```
步驟：
1. 點擊發文輸入區域（"你在想些什麼?" 文字框）
2. 輸入優化後的文章內容
3. 定位「發佈」按鈕
4. 點擊發佈
5. 等待 2 秒確保發佈完成
6. 截圖確認發佈成功
```

### 第 5 步：確認和反饋
- 顯示最終發佈的文章內容
- 確認文章已出現在時間線上
- 說明用戶接下來可以在 Facebook 上看到的內容

## 核心工具和命令

### 使用的 Chrome 工具

| 工具 | 用途 | 範例 |
|------|------|------|
| `find` | 尋找頁面元素 | `find("post text input area")` |
| `computer` | 執行點擊、輸入、截圖 | `computer("left_click", ref=ref_id)` |
| `navigate` | 導航到 URL | `navigate("https://facebook.com")` |
| `wait` | 等待指定秒數 | `wait(2)` |

### 常見的 Facebook 元素參考

- **發文輸入框**：通常標題為「在想些什麼?」或類似文本
- **發佈按鈕**：標題為「發佈」（中文）、「Post」（英文）

## 常見情況和解決方案

### 情況 1：Chrome 擴展連接失敗
**症狀**：「Chrome extension is not connected」
**解決**：
- 確保 Chrome 瀏覽器已打開
- 檢查擴展是否已啟用
- 在瀏覽器切換到 Claude 標籤頁面

### 情況 2：找不到發文輸入框
**症狀**：find 工具無法定位到文字框
**解決**：
- 使用 `read_page` 查看完整頁面結構
- 確認 Facebook 是否已完全加載
- 嘗試重新整理頁面

### 情況 3：發佈按鈕點擊失敗
**症狀**：按鈕點擊後沒有反應
**解決**：
- 確認文本框內容完整（沒有未完成的標籤或引用）
- 等待 1-2 秒後重試
- 檢查是否有彈窗阻擋

## 最佳實踐

### 文章撰寫建議

✅ **推薦做法：**
- 開頭使用能夠吸引注意的表情符號
- 清晰的段落分隔（使用換行符）
- 包含具體的例子或數據
- 結尾加入行動呼籲或個人簽名

❌ **避免做法：**
- 過長的文本塊（應分成多個段落）
- 太多同時發佈相同內容
- 包含敏感信息未經驗證

### 發佈時機

- 考慮你的受眾活躍時段
- 避免在深夜或早晨發佈（除非有特殊原因）
- 定期發佈有助於建立受眾互動

## 故障排除檢查清單

在要求幫助前，檢查以下事項：

- [ ] Chrome 瀏覽器已打開
- [ ] 已登入 Facebook 帳戶
- [ ] Claude in Chrome 擴展已啟用
- [ ] 網路連接穩定
- [ ] 文章內容已準備好或提供了主題大綱
