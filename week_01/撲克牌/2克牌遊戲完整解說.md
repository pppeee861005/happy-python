這個專案分為兩個檔案：

  ---
  Card.py（模組）— 定義三個類別

  1. Card（一張牌）

  class Card:
      SUITES = {1: 'Club', 2: 'Diamond', 3: 'Heart', 4: 'Spade'}
      RANKS  = {11: 'Jack', 12: 'Queen', 13: 'King'}
  - 花色用數字 1~4 儲存，數字越大花色越大
  - 特殊點數（11、12、13）顯示為 Jack、Queen、King

  ┌─────────┬─────────────────────────────┐
  │  方法   │            功能             │
  ├─────────┼─────────────────────────────┤
  │ __str__ │ 印出 "7 of Club" 這樣的格式 │
  ├─────────┼─────────────────────────────┤
  │ __gt__  │ 支援 card1 > card2          │
  ├─────────┼─────────────────────────────┤
  │ __lt__  │ 支援 card1 < card2          │
  ├─────────┼─────────────────────────────┤
  │ __eq__  │ 支援 card1 == card2         │
  └─────────┴─────────────────────────────┘

  ---
  2. Deck（整副牌）

  self.cards = [Card(suite, rank)
                for suite in range(1, 5)
                for rank  in range(1, 14)]
  random.shuffle(self.cards)
  - 用雙層迴圈建立 4 花色 × 13 點數 = 52 張牌
  - 建立後立即洗牌

  ┌────────────┬───────────────────────────────────────────┐
  │    方法    │                   功能                    │
  ├────────────┼───────────────────────────────────────────┤
  │ deal()     │ 從牌堆取出一張（用 pop()），空了回傳 None │
  ├────────────┼───────────────────────────────────────────┤
  │ is_empty() │ 判斷牌堆是否為空                          │
  └────────────┴───────────────────────────────────────────┘

  ---
  3. Hand（玩家手牌）

  ┌────────────────┬──────────────────┐
  │      方法      │       功能       │
  ├────────────────┼──────────────────┤
  │ add_card(card) │ 把一張牌加入手牌 │

  ---
  撲克牌.py（主程式）— 遊戲邏輯

  流程圖

  輸入兩位玩家名字
         │
         ▼
    建立牌堆 (52張洗好的牌)
         │
         ▼
  ┌─── 牌堆有牌？ ───否───► Game Over
  │         │是
  │         ▼
  │    各發一張牌
  │         │
  │    card1 > card2 ──► p2 收走兩張
  │    card2 > card1 ──► p1 收走兩張
  │    card1 == card2 ──► 各自保留
  │         │
  └─────────┘
         │
         ▼
    比較手牌張數
    張數少的人贏！

  勝負判斷

  if card1 > card2:
      p2.hand.add_card(card1)  # 贏家收走兩張
      p2.hand.add_card(card2)
  elif card2 > card1:
      p1.hand.add_card(card1)
      p1.hand.add_card(card2)
  else:
      p1.hand.add_card(card1)  # 平手各自保留
      p2.hand.add_card(card2)

  ▎ 注意遊戲規則：贏家把自己的大牌交給輸家，所以輸家手牌越來越多。最後手牌少的人獲勝！
